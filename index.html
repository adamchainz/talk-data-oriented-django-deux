1. Layout

Problem: find mean of 10,000 2D points

OOP / Row-oriented


    class Point:
        def __init__(self, x, y):
            self.x = x
            self.y = y

    from random import random
    points = [
        Point(random(), random())
        for _ in range(10_000)
    ]
  

    def mean_point():
        return Point(
            sum(p.x for p in points) / len(points),
            sum(p.y for p in points) / len(points),
        )
